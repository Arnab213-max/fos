Food Ordering System - CRUD Operation
A complete food ordering system with full CRUD (Create, Read, Update, Delete) functionality. This project demonstrates a modern web application using HTML, CSS, JavaScript for the frontend, and PHP with MySQL for the backend API.

http://localhost/fos/ - for a dashboard and a workflow

 ✨Features✨
Create - Add new food items with name, price, category, and description

Read - View all food items in an organized dashboard

Update - Edit existing food details

Delete - Remove food items from the menu

Responsive Design - Works on desktop, tablet, and mobile

Category Filtering - Filter items by category (Burgers, Pizza, Sushi, Drinks, Desserts)

Real-time Updates - Instant UI updates without page refresh

Search Functionality - Quick search through menu items


🛠️Technologies Used 🛠️
Frontend:
HTML5
CSS3 (Flexbox, Grid, Animations)
JavaScript (ES6+)
Font Awesome Icons

Backend logic used in it :
PHP 7.4
MySQL Database

📋 Prerequisites 📋
Before running this project, make sure you have:

PHP 7.4 or higher installed

MySQL 5.7 or higher

Web server (Apache, Nginx, or PHP built-in server)
 A Modern web browser
